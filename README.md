# Support Vector Machines and Naive Bayes

## Overview

This repository contains practical implementations of Support Vector Classification (SVC) and Naive Bayes using Scikit-learn. It demonstrates model training, preprocessing, kernel comparison, feature encoding, hyperparameter tuning, and evaluation techniques.

---

## Algorithms Included

### Support Vector Classifier (SVC)

- Linear Kernel
- Polynomial Kernel
- Sigmoid Kernel
- Kernel Comparison
- GridSearchCV
- Model Evaluation

### Naive Bayes

- Data Preprocessing
- Label Encoding
- Train-Test Split
- Model Training
- Prediction
- Performance Evaluation
